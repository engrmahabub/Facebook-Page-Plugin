== Description ==
Facebook Page Plugin lets you easily embed and promote any Facebook Page on your website. Just like on Facebook, your visitors can like and share the Page without leaving your site.

= Features =
* Show or hide page cover photo.
* Show or hide page posts.
* Show or hide friend's faces.
* Small or large header.
* Width adapt to container width.
* Show or hide action to call button.
* Tabs show or hide.
* Width and height set option, it is not necessery.
* The Facebook App ID set if you have or take it blank.

== Installation ==
1. Upload the entire `facebook-page-plugin` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now Facebook Page Plugin widget abailable in your widget section.

= Usage =

1. Go to Appearance > Widgets.
2. SM Facebook Page widget.
3. Enter widget title
4. Enter Facebook App ID if you have;
5. Enter Facebook Page Url.
6. Check of uncheck options.
7. Set width and height if you need.
8. Hit the save button.
